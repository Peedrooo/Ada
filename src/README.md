# Ada
Algoritmo gerador de grade horária para a Universidade de Brasilia

## Como executar

### Requisitos
- Python 3.11.x
- Pip 21.3

### Ambiente virtual
```bash
Ada> python -m venv .venv
Ada> .venv\Scripts\activate
```

### Dependências
```bash
(.venv) Ada> pip install -r src/requirements.txt
```
